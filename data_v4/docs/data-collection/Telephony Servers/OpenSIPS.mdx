# OpenSIPS collector

## Configuration

If using [our official native DEB/RPM packages](/docs/installing/native-linux-distribution-packages), make sure `netdata-plugin-chartsd` is installed.

Edit the `charts.d/opensips.conf` configuration file using `edit-config` from the Netdata [config
directory](/docs/configuring/configuration), which is typically at `/etc/netdata`.

```bash
cd /etc/netdata   # Replace this path with your Netdata config directory, if different
sudo ./edit-config charts.d/opensips.conf
```
